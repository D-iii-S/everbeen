Meaning of some weird symbols:

	M:	mandatory parameter
	O:	optional parameter
	F:	(optional) flag
	BLOB:	a blob (standard/file input) parameter
	*:	any string

The following tree is structured like this:

	module
		action
			parameter
			flag
